# Plan 02-02 — Targeted optimizations & regression smoke

## Objective

На основе `PERF-BASELINE.md` внести **ограниченный набор** измеримых оптимизаций (CPU и/или wgpu), не удаляя режимы рендера. Подтвердить отсутствие регрессий: `cargo test`, сборка release, ручной смоук (2D, 3D, path trace при наличии GPU, кэш сканов, LoD merge/expand).

## must_haves

- `cargo test` — exit 0.
- `cargo build --release` — exit 0.
- Краткий список изменений в `PERF-BASELINE.md` или новом `PERF-CHANGES.md` (что сделано, ожидаемый эффект).
